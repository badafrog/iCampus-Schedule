 Sungkyunkwan Univ. 2021-2, 문제해결과알고리즘, LearningFair Team I8

* 주의사항
1. Windows 에서만 실행
2. chrome 최신 버전 필요. 2021/11/18 16:46 기준 v96
3. iCampus Schedule.py 가 프로그램 실행 파일.
4. 폴더 안의 파일/폴더 위치 절대 변경 금지

# 필수 라이브러리
라이브러리 설치가 안되어있다면
Command Prompt 에서 'pip install (라이브러리)' 로 설치
1. tkinter ('pip install tk')
2. tkcalendar ('pip install tkcalendar')
3. selenium ('pip install selenium')
4. chromedriver-autoinstaller ('pip install chromedriver-autoinstaller')
5. pytz ('pip install pytz')
6. requests ('pip install requests')

# chrome, chromedriver 버전 확인
1. chrome 설치 필수
2. chrome 버전은 최신버전으로 업데이트
3. chromedriver.exe 는 프로그램 실행 시 다운 여부 확인
4. chromedriver.exe 가 없다면 경고창에서 다운받을 것인지 확인
5. chromedriver.exe 자동 설치
